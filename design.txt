Title: Math Painter

Description:
An app that lets the user provide the start coordinates of
geometrical shapes such as squares and rectangles,
their dimentions, and their colors, and th program produces
an image file canvas with the geometrical shapes drawn in it.

Objects:
-- Square:
    - x
    - y
    - side
    - color
    - draw(canvas)
-- Rectangle:
    - x
    - y
    - width
    - height
    - color
    - draw(canvas)
-- Canvas:
    - width
    - height
    - color
    - make(image_path)
